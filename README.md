# MSFT Performance Analysis (Power BI)

## Project Overview
This project presents an interactive Power BI dashboard analyzing Microsoft (MSFT) financial and stock performance.
The goal is to understand trends, key financial metrics, and overall company performance using historical data.

## Tools & Technologies
- Power BI
- Microsoft Excel
- Financial & stock market data

## Key Dashboard Insights
- Revenue and profit trend analysis
- Stock price movement over time
- Key performance indicators (KPIs)
- Year-over-year performance comparison

## Files in This Repository
- `MSFT Report.pbix` – Interactive Power BI dashboard
- `MSFT Report.pdf` – Exported dashboard for quick viewing
- `MSFT.xlsx` – Dataset used for analysis

## Business Value
This dashboard helps stakeholders quickly evaluate Microsoft’s financial health and identify performance trends to support data-driven decisions.
